## v1.0.17

[compare changes](https://github.com/Meta-CTO/strapi-plugin-firebase-auth/compare/v1.0.16...v1.0.17)

- Added password reset functionality with email support
- Implemented email service for sending password reset links
- Added customizable email templates for password resets
- Enhanced Firebase configuration management with better error handling

### Improvements
- Improved user management UI components
- Refactored forms with better validation
- Enhanced pagination and table components
- Better TypeScript type definitions
- Code formatting improvements with Prettier

### Bug Fixes
- Fixed password provider detection logic
- Resolved issues with user form submissions
- Fixed validation for required fields
